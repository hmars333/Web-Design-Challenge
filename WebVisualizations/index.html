Summary:

This Project analyzes various cities using the OpenWeatherMap API to see how weather changes as you near the Equator. I used the OpenWeatherMap API to gather data from 500 random cities. I then used Python's Matplotlib to plot the data and visualize my findings. I plotted the Temperature, Humidity, Cloudiness and Windspeed to meausre different factors. This website includes the dataset and the plots for each of these factors with descriptions of my findings.

Use the Plots menu to navigate through each chart. The Comparison page will display all charts, the Data page shows the dataset used.
